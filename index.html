如何利用github和hexo搭建个人博客

Veröffentlicht am 2017-07-19
  • 为什么想到搭建博客

暑假想自己学一点java和python，突然想到在学之前何不先学习搭建一个博客，之后学习过程中有什么心得体会也好分享一下，顺带记录一下自己每天的学习经历，于是乎便有了这篇文章。

  • 花了多长时间搭建自己的博客

大概花了一天多的时间吧，看着别人的教程，一点点地跟着做，中间也遇到了许多问题，去查了很多次资料才慢慢解决，总之一名优秀的程序员一定要学会自己独立去搜索自己需要的东西。

  • 搭建博客使用的是hexo+GitHub

hexo是一个快速高效的博客框架，使用Markdown解析文章，几秒内就可以自动生成静态网页，可以理解成一个高效的辅助编码的工具。而GitHub是一个通过Git进行版本控制的软件源代码托管服务，在搭建博客过程中可以理解成给自己搭建的博客提供一个平台以供他人访问。

一、配置GitHub

首先注册、登录github.com
之后点击右上的加号，选择New repository，之后进入新的页面，Repository name填写yourname.github.io,比如我的用户名是overtansy，那填写的就是overtansy.github.io（之后这个就是你建成的博客的域名了），下方点击创建，然后就暂时不用管这个了。

二、环境安装（node、git）

hexo需要Node.js和git两个必备应用程序。git是一款版本控制软件，Node.js则是一款JS运行平台，想要更加深入的了解这两款应用请自行在网络上搜索。下面附上下载链接：

三、hexo安装

首先附上hexo官方教程的链接，我在学习搭建博客过程中这个教程帮了非常大的忙，有能力者其实可以直接按照官方教程学习搭建自己的博客。

在装好了git和Node.js后可以直接通过npm安装Hexo，在开始菜单里找到git，然后打开git bash，输入$ npm install -g hexo -cli，接下来慢慢等待即可，安装好hexo后需要的搭建环境就全部装好了，下面就可以开始搭建自己的博客了。

四、搭建本地服务站

建一个文件夹，在文件夹里面点击右键git bash here，之后会出现git命令框，输入$ hexo init blog（建立初始站），等待一会儿，发现文件夹里面多了很多东西：

  1. .deploy_git文件夹，这个是用来存放部署后的网站的
  2. public文件夹，这个是用来存放生成后的网站
  3. scaffolds文件夹，这个用来存放模板，默认有draft,page,post三种
  4. source文件夹，用于存放资源，比如文章图片之类的
  5. themes文件夹，用于存放主题

接着在git命令框中输入$ hexo generate（生成静态网页文件），由于初始化的hexo自带一篇hello world文章，所以执行此命令后就可以生成此文章的静态网页文件，之后输入$ hexo server（启动本地服务器），打开浏览器输入http://localhost:4000 就可以看到这篇文章了，但此时还没有发布到Github上所以别人还看不到。

五、ssh密匙

ssh密匙可以让我们方便的登录到服务器上而不需要输入密码，并且更加安全，所以在发布到GitHub上之前首先设置一下ssh密匙，按ctrl+c取消启动本地服务器，之后输入ssh-keygen -t rsa -C "Github的注册邮箱"一直按回车直到显示your public key has been saved “一个地址”，找到该文件，用sublime test打开（没有的话，这里是下载链接），复制里面的所有内容，然后进入之前的github.com页面，点击右上角的头像选择settings，在左边找到ssh and gpg keys并单击，右上选择new ssh key，title输入blog，key里面粘贴刚刚复制的内容，然后点击下面的add ssh key，至此ssh就已经设置好了。

六、配置博客

在之前建站的文件夹里找到config.yml用sumlime text打开，这里面有许多的信息，首先最上面的#Site下面是网站的基础信息，可以自行进行设置，注意填写信息时“：”后要加空格

  • title：名字
  • subtitle：副标题
  • description：描述，用于给搜索引擎看
  • author：作者
  • language：语言
  • timezone：时区

在最下方找到deploy，tpye填写git，然后在下面继续添加，yourname处填写之前github的名字

repo:https://github.com/yourname/yourname.github.io.git

branch:master

然后ctrl+c保存，在git命令框中输入$ hexo generate（生成静态网页），$ hexo deploy（部署至网页上），期间会出现github的登录，填写自己的用户名和密码就行了，这个时候不出意外你的博客就应该已经搭建好了，输入yourname.github.io就可以进入自己的博客了。

七、发表文章

在git命令框中输入$ hexo new "文章名字"，然后在source文件夹中的posts文件夹里面找到新建的这篇文章，用markdown语法进行编辑，好了之后保存，在git命令框中执行$ hexo clean（清理缓存），再执行$ hexo g -d(生成静态网页并部署)，打开自己的博客，就可以看到自己的文章了。

八、总结

在实际搭建过程中可能会遇到各种问题，这个时候就要多搜索，仔细观看hexo的官方教程，毕竟很难有什么事情是能一次性很顺利的解决的，唯有不停搜不停的思索才能真正掌握一些东西，最后附上一些常见问题的链接，希望能有所帮助
